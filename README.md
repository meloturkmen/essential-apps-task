1️⃣ Clone this repository

git clone https://github.com/meloturkmen/essential-apps-task.git

cd essential-apps-task

2️⃣ Install dependencies

npm install

3️⃣ Run the app

npm run dev

4️⃣ Open the inventory page

Navigate inside the app to Inventory Overview (route: /app/products).

Use the filter buttons (All / Low / Out) and adjust the Low stock threshold to test different views.
